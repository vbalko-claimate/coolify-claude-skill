# Coolify Claude Skill

A Claude Code skill for managing Coolify deployments, applications, and infrastructure.

## Features

- Deploy and manage applications
- Control databases and services
- Monitor logs and status
- Infrastructure management via SSH

## Installation

```bash
# Clone to Claude skills directory
git clone https://github.com/vbalko-claimate/coolify-claude-skill.git ~/.claude/skills/coolify
```

## Usage

```bash
# In Claude Code
/coolify deploy my-app
/coolify logs my-app
/coolify status
```

## Requirements

- Claude Pro/Max/Team/Enterprise
- Access to Coolify instance
- Coolify API token
